# TrainingTimer

An Android app with a timer that automatically resets and alternates between 60 seconds and 90 seconds.

## Features

- Timer automatically resets when it reaches zero
- Alternates between 60-second and 90-second intervals
- Start/Stop button to control the timer
- Clean, modern UI with large timer display

## How to Build

1. Open the project in Android Studio
2. Sync Gradle files
3. Run the app on an emulator or physical device

## Requirements

- Android Studio (latest version recommended)
- Minimum SDK: 24 (Android 7.0)
- Target SDK: 34

## Usage

The timer starts automatically when the app launches. It will:
1. Count down from 60 seconds
2. When it reaches 0, automatically reset to 90 seconds
3. When it reaches 0 again, automatically reset to 60 seconds
4. Continue alternating between 60s and 90s intervals

You can use the Start/Stop button to pause and resume the timer.
