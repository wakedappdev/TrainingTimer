# Training Timer App

A highly configurable interval timer app for Android, perfect for workouts, studying, and other timed activities.

## Features

- **Alternating Intervals:** Set two different interval durations (in seconds) that automatically alternate.
- **Configurable Timers:** Easily change the length of each interval and the total session duration directly within the app.
- **Total Session Timer:** Set a total workout time (in minutes), and the app will automatically stop when the time is up.
- **Rest Button:** Take a quick 30-second break at any time. After the rest, the timer will resume its previous interval cycle.
- **Start/Stop Control:** Full control to pause and resume your session.
- **Elapsed Time Display:** Keep track of your total workout time, displayed above the control buttons.
- **Audio Cues:** A low-volume beep signals the end of each interval.
- **Screen Always On:** The device's screen will not turn off while the app is active, so you can always see your progress.

## How to Use

1.  **Set Your Intervals:**
    -   Enter your desired duration (in seconds) for the first interval in the **1st interval time** field.
    -   Enter the duration for the second interval in the **2nd interval time** field.
2.  **Set Total Duration:**
    -   Enter the total time you want the session to last (in minutes) in the **Stop after X mins** field.
3.  **Save Settings:**
    -   Press the **Save** button to apply your new settings. The timer will automatically restart with the new configuration.
4.  **Control the Timer:**
    -   The timer starts automatically when the app launches.
    -   Use the **Start/Stop** button to pause and resume the timer.
    -   Use the **Rest** button to start a 30-second rest period.
